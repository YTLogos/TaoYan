Circos配置文件详解

Circos的配置文件准备

Circos的使用主要通过输入一个配置文件，改配置文件的主要内容以各种区块表示，大区块中可以包含小区快。区块中以"变量=值"的方式来进行参数的设定：

<links>
<link>
  file = data/set1.txt
  color = black
  ...
</link>
<link>
  file = data/set2.txt
  color = red
  ...
</link>
</links>
